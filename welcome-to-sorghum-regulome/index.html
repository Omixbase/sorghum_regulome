Welcome to sorghum regulome

The sorghum transcriptional data set was generated by integrating 873 public available RNA-seq data sets. The sorghum transcriptional dataset provides gene expression landscape across a diverse set of 19 tissues from underground (roots) to above ground. Other than gene expression across tissues, we also included the gene diversity or evolution information. Hope this data can facilitate sorghum functional genomics discovery.

This gene expression calling was based on the sorghum reference genome version genome 3.1 (Cooper et al. 2019 BMC Genomics volume 20). The alignment was performed using STAR (Dobin et al. 2013 Bioinformatics 29(1):15-21). The gene expression was estimated from bam file using StringTie (Pertea et al. 2015 Nature Biotechnology 33:290-295). The transcription data sets included 31,541 genes and 19 tissues.
